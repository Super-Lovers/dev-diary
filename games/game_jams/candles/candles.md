### Candles
##### gamejam desktop all

In Candles, You are a girl called Jane that is struggling with an  unfortunate relationship with her family and to cope, she decides it is  time to go, and so your adventure to escape begins.

This game was made for the Global Game Jam 2019 in two days and then updated after the jam.

#### Controls

 [WSAD] or Arrow keys for movement. 
Key [E] to interact with in-game objects. 

#### Installation instructions
Extract the .ZIP file to a new directory or currently opened directory and run the executable "ggj" file.
